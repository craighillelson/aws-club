# RDS
RDS (Relational Database Service) is AWS's managed relational database service.

**What it does:**

- **Managed databases** - Supports MySQL, PostgreSQL, MariaDB, Oracle, SQL Server, and Aurora
- **Automated maintenance** - Handles backups, patching, monitoring
- **High availability** - Multi-AZ deployments for automatic failover
- **Read replicas** - Scale read-heavy workloads across multiple copies
- **Point-in-time recovery** - Restore to any second within your retention period
- **Encryption** - At rest and in transit

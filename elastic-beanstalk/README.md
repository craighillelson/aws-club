# Elastic Beanstalk

Elastic Beanstalk is AWS's platform-as-a-service for deploying web applications.

**What it does:**

- **Automatic provisioning** - Deploys EC2, ELB, Auto Scaling, CloudWatch based on your app needs
- **Multiple platforms** - Supports Java, .NET, PHP, Node.js, Python, Ruby, Go, Docker
- **Simplified deployment** - Upload code via console, CLI, or Git
- **Managed updates** - Handles platform patching and maintenance
- **Retain control** - You can still access and modify underlying resources if needed
- **Environment management** - Easy blue/green deployments, separate dev/staging/prod

# CloudFront
CloudFront is AWS's content delivery network (CDN) service.

**What it does:**

- **Global content distribution** - Caches content at edge locations worldwide (200+ locations)
- **Reduced latency** - Serves content from the location nearest to users
- **Origin flexibility** - Works with S3, EC2, ELB, or any custom HTTP server
- **Security features** - Integrates with AWS Shield (DDoS protection), WAF, and SSL/TLS certificates
- **Dynamic and static content** - Caches static files, accelerates dynamic content delivery
- **Streaming support** - Handles video streaming and live events

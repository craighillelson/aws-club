# CloudWatch
CloudWatch is AWS's monitoring and observability service.

**What it does:**

- **Metrics** - Collects performance data (CPU usage, disk I/O, network traffic, etc.)
- **Logs** - Aggregates application and system logs in one place
- **Alarms** - Triggers notifications or actions when metrics cross thresholds
- **Dashboards** - Visualize your metrics in real-time
- **Events/EventBridge** - React to changes in your AWS environment
